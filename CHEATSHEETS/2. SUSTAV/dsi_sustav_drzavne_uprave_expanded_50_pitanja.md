# DSI Sustav Državne Uprave Expanded + 150 Pitanja

Proširena radna verzija uz `dsi_sustav_drzavne_uprave_cheat_sheet.md`.

Namjena:
- dodatno širenje gradiva
- 150 mogućih pitanja s kratkim odgovorima
- fine formulacije i klasične zamke

## Što dodatno učiti

Posebno razlikuj:
- `ministarstvo / državna upravna organizacija / upravna organizacija u sastavu ministarstva`
- `zakon / uredba o unutarnjem ustrojstvu / pravilnik o unutarnjem redu`
- `provedbeni propis / naredba / interni opći akt`
- `državni dužnosnik / rukovodeći državni službenik / namještenik`
- `provedba politike / neposredna provedba zakona / upravni nadzor / inspekcijski nadzor`

## Fina pitanja i zamke

- ministarstvo se osniva `zakonom`, ne uredbom
- državna upravna organizacija `jest` tijelo državne uprave
- upravna organizacija u ministarstvu `nije` tijelo državne uprave
- ministra može zamijeniti samo `državni tajnik`
- `glavni tajnik` ministarstva je službenik, ne dužnosnik
- `pravilnik o unutarnjem redu` nije provedbeni propis
- `naredba` je podzakonski propis, ali nije provedbeni propis
- `provedba državne politike` i `upravni nadzor` ne povjeravaju se
- podruzne jedinice ne mogu biti u sjedistu tijela
- o sukobu nadležnosti tijela državne uprave odlučuje `Vlada`
